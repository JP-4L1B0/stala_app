# STALA - Sheet Music to Guitar Tablature Translator

STALA is a mobile-based music translation application that converts grand staff sheet music into guitar tablature using Optical Music Recognition (OMR) and pitch-based tablature generation. The application is developed using Flutter and designed to run efficiently on Android devices.

## Features

* Import or capture sheet music images
* Detect and recognize music symbols using AI-based object detection
* Interpret musical notes from grand staff notation
* Convert detected notes into guitar tablature
* Generate lightweight mobile-compatible tablature outputs
* Export generated tablature results
* User-friendly mobile interface optimized for Android devices

## Technologies Used

* Flutter
* Dart
* ONNX Runtime
* Computer Vision / Optical Music Recognition (OMR)
* Android Studio

## Getting Started

### Prerequisites

Before running the project, make sure the following are installed:

* Flutter SDK
* Android Studio
* Android SDK
* Git

### Installation

1. Clone the repository

```bash
git clone <repository-link>
```

2. Navigate to the project directory

```bash
cd stala_app
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
